| l4_disc1 | S_DISCOVERY (Зов Разведчика) | 2 | +1 |
| l4_disc2 | S_HEALTHCHECK (Сердцебиение) | 2 | 0 |

**Активные долги**: нет (D_NEEDS_HEALTHCHECK закрыт, D_NO_OBSERVABILITY закрыт, D_LOG_VOLUME закрыт)
**Capabilities**: HAS_TLS, HAS_AUTHZ, HAS_FAILOVER, HAS_LOGS, HAS_LOG_FILTERING

**Что понимает игрок**: самый прямой путь. S_LOGGING (Перо Летописца) обязателен для закрытия D_NO_OBSERVABILITY; S_LOG_FILTER (Сито Смыслов) закрывает долг S_LOGGING (Перо Летописца).

## Путь B — Аудит + Канарейка (S_ACCESS_LOGS (Книга Доступа) + R_CANARY (Канарейка в шахте))

Альтернативный путь наблюдаемости: ACCESS_LOGS создаёт D_ACCESS_LOG_VOLUME (некритический, +2 ауры), R_CANARY (Канарейка в шахте) компенсирует -1. Суммарная аура = 3 = лимит — проходит на грани.

| Слот | Карта | Монеты | Аура |
|---|---|---|---|
| l4_auth | S_AUTHZ (Страж Порогов) | 3 | 0 |
| l4_tls | S_TLS_CERT (Печать Серта) | 2 | 0 |
| l4_disc1 | S_DISCOVERY (Зов Разведчика) | 2 | +1 |
| l4_disc2 | S_HEALTHCHECK (Сердцебиение) | 2 | 0 |
| l4_logs | S_ACCESS_LOGS (Книга Доступа) | 3 | +1 |
| l4_obs_rit | R_CANARY (Канарейка в шахте) | 1 | −1 |
| Некрит. F1_OVERFLOW | — | — | +2 |
| **Итого** | | **13** | **3** |

**Активные долги**: D_ACCESS_LOG_VOLUME → F1_OVERFLOW (некритический)
**Capabilities**: HAS_TLS, HAS_AUTHZ, HAS_FAILOVER, HAS_ACCESS_LOGS
**DoD**: HAS_TLS ✓, HAS_AUTHZ ✓, HAS_FAILOVER ✓
**Итог**: PASS (13 монет / 3 ауры = лимит)

**Что понимает игрок**: S_ACCESS_LOGS (Книга Доступа) создаёт нерастворимый долг на этом уровне. Без обряда аура уходит за лимит. R_CANARY (Канарейка в шахте) — единственный способ удержаться. Компромисс: audit-лог дороже и требует обряда.
